notifications

Receive notifications for activity on your account or statuses.

get
Get all notifications

https://mastodon.example/api/v1/notifications

Notifications concerning the user. This API returns Link headers containing links to the next/previous page. However, the links can also be constructed dynamically using query params and id values.

Returns: Array of Notification
OAuth: User token + read:notifications
Version history:
0.0.0 - added
2.6.0 - add min_id
2.9.0 - add account_id
3.1.0 - add follow_request type

Żądanie

Nagłówki
Authorization
wymagane
string
Bearer <user token>
Parametry zapytania
max_id
nieobowiązkowe
string
Return results older than this ID
since_id
nieobowiązkowe
string
Return results newer than this ID
min_id
nieobowiązkowe
string
Return results immediately newer than this ID
limit
nieobowiązkowe
string
Maximum number of results to return (default 20)
exclude_types
nieobowiązkowe
array
Array of types to exclude (follow, favourite, reblog, mention, poll, follow_request)
account_id
nieobowiązkowe
string
Return only notifications received from this account
