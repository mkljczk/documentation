post
Create a filter

https://mastodon.example/api/v1/filters

Returns: Filter
OAuth: User token + write:filters
Version history:

  • 2.4.3 - added

Żądanie

Nagłówki
Authorization
wymagane
string
Bearer <user token>
Parametry formularza
phrase
wymagane
string
Text to be filtered
context
wymagane
array
Array of enumerable strings home, notifications, public, thread. At least one context must be specified.
irreversible
nieobowiązkowe
boolean
Should the server irreversibly drop matching entities from home and notifications?
whole_word
nieobowiązkowe
boolean
Consider word boundaries?
expires_in
nieobowiązkowe
integer
Number of seconds from now the filter should expire. Otherwise, null for a filter that doesn’t expire.

Odpowiedź

200: Success

The newly-created filter will be returned. 

{
  "id": "8449",
  "phrase": "test",
  "context": [
    "home",
    "notifications",
    "public",
    "thread"
  ],
  "whole_word": false,
  "expires_at": "2019-11-26T09:08:06.254Z",
  "irreversible": true
}

401: Unauthorized

Invalid or missing Authorization header 

{
  "error": "The access token is invalid"
}

422: Unprocessable Entity

If phrase or context are not provided properly

phrase

{
  "error": "Validation failed: Phrase can't be blank"
}
context
{
  "error": "Validation failed: Context can't be blank, Context None or invalid context supplied"
}
